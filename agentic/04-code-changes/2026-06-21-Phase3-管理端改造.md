## 与用户端 Phase 2 联调约定

- 用户端双写 `paleo_admin_*` ↔ 管理端读取
- 管理端审核（入会/退会/凭证/发票/延期）同步 `paleo_*` 用户端 key
- 统计与导出均走 `collectConferenceAttendees`，按 `confirmed`/`active`/`approved_invoice` 实收状态聚合
