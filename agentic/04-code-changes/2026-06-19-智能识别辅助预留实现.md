# 智能识别辅助预留实现

## 实现范围

- 新增凭证/发票识别结果表：
  - 支持关联会员费缴费记录和会议报名记录。
  - 保存文件角色、文件地址、识别状态、置信度、识别金额、发票号、原始结果和人工复核标记。
- 新增识别服务：
  - 当前默认 provider 为 `MANUAL_FALLBACK`。
  - 未配置真实 OCR 服务时生成 `MANUAL_REQUIRED` 结果，保留人工审核兜底。
  - 后续接外部 OCR/API 时，可替换服务实现并写入真实识别字段。
- 新增后端接口：
  - 会员费凭证/发票识别。
  - 会议报名凭证/发票识别。
  - 识别结果列表查询。
- 管理端增强：
  - 会员费和会议报名文件列增加识别入口。
  - 新增凭证/发票识别结果表，供审核人员参考。
- TODO 文档维护：
  - 已确定主干需求无阻断 TODO。
  - 真实外部 OCR/API Provider 标记为后续优化项。
