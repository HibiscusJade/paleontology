## 变更范围

### 后端服务

- 新增 `paleo_member_profile` 会员状态档案表。
- 新增会员档案 Domain、Mapper、Service。
- 入会申请审核通过后，自动创建或更新会员档案为 `PENDING`。
- 会员费终审确认为 `CONFIRMED` 后，自动激活会员档案为 `ACTIVE`，同步会员类型、有效期和最近缴费记录。
- 退会申请审核通过后，自动更新会员档案为 `WITHDRAWN`。
- 查询会员档案时会按有效期刷新 `EXPIRED` 状态。
- 新增管理端会员档案列表接口和用户端我的会员档案接口。

### 后台管理前端

- 确定需求看板新增“会员状态档案”表格。
- 可查看用户 ID、会员状态、会员类型、有效期、最近申请、最近缴费和更新时间。

### 用户前端

- 我的服务接口新增会员档案查询。
- 个人中心会员状态优先展示会员档案状态。
