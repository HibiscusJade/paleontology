## 变更范围

### 用户前端

- 新增 `src/api/paleo.js`，使用原生 `fetch` 封装用户侧自助接口。
- 支持读取本地 token 并自动追加 `Authorization` 请求头。
- 新增个人中心“我的服务”工作台：
  - 登录、注册和验证码加载。
  - 学会/分会绑定与解绑。
  - 按绑定关系加载可报名会议。
  - 查询会议公开通知、盖章通知和摘要模板下载权限。
  - 提交我的会议报名。
  - 提交我的入会/退会申请。
  - 提交我的会员费缴费记录。
  - 查看我的会员申请、会员费、会议报名状态。
  - 提交我的参会资料：报告类型、报告题目、住宿、野外阶段和路线。
  - 上传入会/退会申请书。
  - 上传会员费凭证和发票。
  - 上传会议费凭证、发票和摘要 Word。
- 未登录或接口不可用时，在页面内显示接口状态，不影响静态内容展示。
- 登录成功后保存 token，后续自助接口自动携带登录态。
