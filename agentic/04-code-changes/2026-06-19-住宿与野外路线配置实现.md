# 住宿与野外路线配置实现

## 实现范围

- 后端：
  - 会议模型新增住宿配置字段，用于记录房型、房间数和床位数口径。
  - 会议模型新增野外路线配置字段，支持每场会议最多 15 条路线。
  - 新增数据库迁移，扩展会议表配置字段。
  - 会议新增/编辑时限制野外路线最多 15 条。
- 管理端：
  - 会议编辑弹窗支持维护住宿配置和野外路线配置。
  - 报名编辑中住宿类型改为固定房型选项。
  - 报名编辑中野外阶段改为固定阶段选项，野外路线从会议配置中选择，并允许临时补充。
- 用户端：
  - 参会资料提交时，如果会议配置了野外路线，则使用下拉选择。
  - 未配置路线时保留原有手输能力。
- TODO 文档：
