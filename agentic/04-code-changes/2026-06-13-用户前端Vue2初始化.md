# 用户前端 Vue2 初始化

## 基本信息

| 字段 | 内容 |
|---|---|
| 日期 | 2026-06-13 |
| 子项目 | `PaleontologicalResearchFront` |
| 目标 | 基于 Vue 2 初始化用户前端，并参考主项目高保真设计基准搭建首页与核心入口 |

## 变更摘要

- 初始化 Vue 2.7 + Vue CLI 5 工程结构。
- 新增用户前端首页，覆盖学会首页、学会简介、党建文化、会员服务、会议服务、组织机构、个人中心概览。
- 按主项目设计基准使用深蓝、暖金、米白、党建红等视觉体系。
- 新增本地抽象化石与地层主题视觉资产，避免使用脊椎动物化石版权图片。
- 页面数据暂以静态配置维护在 `src/data/siteData.js`，便于后续替换为后端接口。
- README 更新为 Vue 2 工程说明和设计约定。

## 关键文件

| 文件 | 说明 |
|---|---|
| `PaleontologicalResearchFront/package.json` | Vue 2 工程依赖与脚本 |
| `PaleontologicalResearchFront/src/App.vue` | 当前用户前端主页面 |
| `PaleontologicalResearchFront/src/data/siteData.js` | 首页与业务入口静态数据 |
| `PaleontologicalResearchFront/src/styles/main.css` | 全局视觉样式和响应式布局 |
| `PaleontologicalResearchFront/public/assets/paleo-hero.svg` | 首页主题视觉资产 |
| `PaleontologicalResearchFront/vue.config.js` | 构建产物相对路径配置 |

## 验证结果

```bash
npm install
npm run build
npm run lint -- --no-fix
```

结果：

- `npm install` 成功，生成 `package-lock.json`。
- `npm run build` 成功。
- `npm run lint -- --no-fix` 成功，无 lint 错误。
