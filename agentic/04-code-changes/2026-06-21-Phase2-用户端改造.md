## 与管理端 AuditWorkbench 联调约定

用户端提交时双写：

- `paleo_membership_application_{email}` ↔ `paleo_admin_membership_application_{email}`
- `paleo_withdrawal_application_{email}` ↔ `paleo_admin_withdrawal_application_{email}`
- `paleo_confs_{email}` ↔ `paleo_admin_confs_{email}`
- `paleo_society_membership_{email}` ↔ `paleo_admin_society_membership_{email}`
- 新注册用户同步至 `paleo_admin_all_users`

管理端 approve/reject 已写回 `paleo_*` key；用户端通过 `storage` 事件 + 定时轮询刷新 React 状态。

## 手动联调路径（建议）

1. website 注册新用户 → 选择「成为正式会员」→ 上传入会申请书  
2. admin AuditWorkbench「入会申请」Tab 出现待审项 → 通过  
3. website 会员服务页自动变为「可缴会费」（轮询 ≤3s）  
4. PersonalCenter「缴费记录」按总学会分组展示凭证/发票
