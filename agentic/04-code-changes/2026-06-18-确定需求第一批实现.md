## 变更范围

### 后端服务

- 新增 `edu.xzit.core.paleo` 业务包。
- 新增学会/分会、用户绑定、会议、会议四类费用、会议报名资料实体。
- 新增 MyBatis-Plus Mapper 和 Service。
- 新增接口：
  - `/paleo/associations`
  - `/paleo/conferences`
  - `/paleo/registrations`
- 新增会议统计能力：
  - 按单次会议统计四类费用、报名人数、报告、住宿、野外信息。
  - 按学会/分会或全局统计会议费金额和笔数。
- 新增 SQL 脚本 `V0.0.2__Paleo_confirmed_requirements.sql`，初始化总学会和 11 个分会。

### 后台管理前端

- 新增 `src/api/paleo/confirmed.js`。
- 新增 `src/views/paleo/confirmed/index.vue`，作为确定需求实现看板和基础数据入口。
- 新增固定路由 `/paleo/confirmed`。

### 用户前端

- 会议数据从会员价/非会员价扩展为四类费用：
  - 学生会员。
  - 非学生会员。
  - 学生非会员。
  - 非学生非会员。
- 前台会议卡片增加缴费后资料下载、摘要、住宿和野外报名规则展示。
- 个人中心区域增加确定需求规则卡片。
