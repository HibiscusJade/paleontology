## 变更范围

### 后端服务

- 新增 `paleo_admin_association` 管理员-学会/分会数据范围表。
- 新增管理员数据范围实体、Mapper、Service 和 Controller。
- 新增 `PaleoStateMachine`，约束核心状态流转：
  - 入会/退会申请只允许从 `PENDING` 审核到 `APPROVED` 或 `REJECTED`。
  - 会员费凭证、发票审核按 `VOUCHER_REVIEW -> VOUCHER_APPROVED -> INVOICE_REVIEW -> CONFIRMED` 等路径约束。
  - 已确认缴费记录不能反复审核。
- 新增业务级上传接口：
  - 会员申请书上传并绑定申请记录。
  - 会员费凭证/发票上传并绑定会员费记录。
  - 会议公开通知、盖章通知、摘要模板上传并绑定会议记录。
  - 会议报名凭证、发票、摘要上传并绑定报名记录。
- 学会/分会、会议、报名、统计接口接入数据范围过滤：
  - 超级管理员可访问全部。
  - 普通分会管理员只能访问 `paleo_admin_association` 中绑定的学会/分会。
  - 未绑定任何学会/分会时返回空业务数据。
