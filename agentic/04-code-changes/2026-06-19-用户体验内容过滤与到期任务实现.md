# 用户体验、内容过滤与到期任务实现

## 实现范围

- 用户端体验增强：
  - 会议资料展示为可点击下载链接，显示文件名。
  - 会议报名根据会员档案自动匹配四类会议费类型。
  - 参会资料补报告类型联动校验：口头/展板必须填写报告题目，仅参会不要求题目。
- 内容按绑定过滤：
  - 新增 `paleo_content` 内容表，支持通知、文章、党建内容。
  - 管理端可维护分会内容，并沿用分会管理员数据范围过滤。
  - 用户端按已绑定学会/分会读取已发布内容，接口不可用时保留静态内容兜底。
- 会员到期任务：
  - 新增会员档案到期扫描方法。
  - 新增 Quartz 任务迁移，每天凌晨扫描到期 ACTIVE 会员并标记为 EXPIRED。
- TODO 文档维护：
  - 除智能识别辅助外，已确定主干需求标记为完成。
  - OCR/外部识别服务保留为最后接入项。
