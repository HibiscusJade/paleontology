## 3. 审核工作台

**页面地址**: 审核工作台  
**可访问角色**: 学会总管理员、财务审核员

管理支付审核的两阶段流程（凭证初审 → 发票终审），适用于会员费和会议费。

### 3.1 凭证初审

对用户上传的缴费凭证进行初步审核。

- 列表包含：邮箱、姓名、类型（会员费/会议费）、金额、提交时间、文件预览、状态
- 单条处理：通过 / 驳回（驳回需填写原因）
- 批量处理：勾选多条后一键通过或驳回
- 点击文件预览可查看凭证图片

<!-- TODO: 截图 — 凭证初审页面 -->

### 3.2 发票终审

对已通过初审、用户已上传发票的申请进行最终审核。

- 列表内容与初审类似
- 单条处理：通过 / 驳回 / 延期（设置新截止日期）
- 同样支持批量处理

<!-- TODO: 截图 — 发票终审页面 -->

**审核状态流转**：

```
提交凭证 → 凭证初审（通过/驳回）→ 上传发票 → 发票终审（通过/驳回/延期）→ 完成
```

## 4. 会员用户管理

**页面地址**: 会员用户管理  
**可访问角色**: 仅学会总管理员

管理所有具备会员资格的用户（包括活跃、待审核、已过期等各种状态）。

- **查找**：可按邮箱/姓名搜索，也可按会员状态、所属分会进行筛选
- **显示**：每页 10 条记录
- **列表信息**：邮箱、姓名、单位、会员状态（彩色标签）、绑定分会、有效期至、操作

### 4.1 查看详情

点击"详情"按钮，右侧滑出信息面板，展示：

- 基本信息（姓名、性别、单位、角色、会员类型、会员状态、有效期等）
- 缴费历史记录
- 通知记录

<!-- TODO: 截图 — 会员列表页 -->

### 4.2 可用操作

- **禁用/启用**：切换会员资格状态
- **手动开通**：将会员直接设为活跃状态，有效期至当年年底

## 6. 会议管理

**页面地址**: 会议管理  
**可访问角色**: 学会总管理员、分会管理员（分会管理员仅可见所属分会的会议）

创建、编辑和管理学术会议。

### 6.1 会议列表

以卡片形式展示所有会议，每张卡片包含：

- 会议名称、状态标签（已发布/草稿）、分会名称
- 地点、日期范围、当前报名人数
- 会员价 / 非会员价

<!-- TODO: 截图 — 会议卡片列表 -->

### 6.2 新建/编辑会议

点击"新建会议"或"编辑"按钮，弹出填写窗口，包含以下内容：

- 会议名称（必填）
- 主办分会（必填，下拉选择）
- 地点（必填）
- 开始/结束日期（必填）
- 会员费（非会员费自动按 +10% 计算）
- 缴费截止日期、摘要截止日期
- 发布状态（草稿 / 已发布）
- 分会场管理（可动态添加/删除多个分会场）

## 8. 财务记录

**页面地址**: 财务记录  
**可访问角色**: 学会总管理员、财务审核员

用于查询所有支付记录的详细信息。

- **查找**：可按类型（全部/会员费/会议费）、状态（10 种状态可选）、日期范围进行筛选
- **显示**：每页 10 条记录
- **列表信息**：记录 ID、用户、类型、金额、状态、凭证日期、发票日期、操作

### 8.1 记录详情

点击查看详情，弹出窗口展示完整信息：

- 用户邮箱、用户名、费用类型、金额
- 目标名称、系统识别金额、提交时间
- 发票截止日期、状态、驳回原因（如有）
- 凭证文件图片预览
- 发票文件图片预览

## 9. 分会管理

**页面地址**: 分会管理  
**可访问角色**: 仅学会总管理员

- **卡片形式**：每个分会一张卡片
  - Logo（图片或默认图标）、名称、分会 ID
  - 简介描述、会员数量标签
  - 已禁用的分会显示"已禁用"标签，卡片半透明
- **编辑**：修改名称、描述、Logo 链接
- **启用/禁用**：点击开关，确认后生效
