## 5. 数据模型补充规范

### 5.1 ConferenceReg / 报名记录扩展字段

```typescript
interface ConferenceRegFields {
  feeType: ConferenceFeeType;      // 报名时锁定
  lockedAmount: number;            // 报名时锁定，OCR/统计用
  identitySnapshot?: {             // 可选，Phase 1 预留
    userType: UserType;
    isStudent: boolean;
    capturedAt: string;
  };
}
```

### 5.2 费用推导唯一入口

```typescript
// 禁止页面直接调用 deprecated getConferenceFee(confId, userType)
const feeType = deriveFeeType(userType, isStudent);
const amount = getConferenceFeeByType(confId, feeType);
// amount === 0 → 关闭该人群报名通道
```

### 5.3 localStorage Key 对齐表

| 语义 | 用户端 Key | 管理端 Key |
|------|------------|------------|
| 会议报名 | `paleo_confs_{email}` | `paleo_admin_confs_{email}` |
| 四类费用配置 | `paleo_conference_fee_configs` | `paleo_admin_conference_fee_configs` |
| 用户类型 | `paleo_user_type_{email}` | `paleo_admin_user_type_{email}` |

**本阶段目标**：admin 审核/统计读取用户端提交的数据时字段一致。
