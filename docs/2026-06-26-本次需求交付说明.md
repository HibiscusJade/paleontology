### 需求 2：退会后再入会流程修复

**问题背景**：用户退会后，点击「重新申请入会」按钮无法跳转到新的入会界面；右侧「会议绑定管理」区域仍显示「您是学会正式会员」，与退会状态矛盾。

**修改结果**：

**用户端 · 学会会员状态卡片：**
- 「重新申请入会」按钮 **恢复可点击**，点击后直接进入入会申请流程第 1 步（下载申请书模板）
- 文案更新为：「如需重新成为会员，点击下方按钮开始重新申请流程。」
- 重新申请流程与首次入会完全一致（提交申请 → 审核 → 缴费 → 激活）
- **无需联系秘书处，无需管理员后台手动解锁**

**用户端 · 会议绑定管理卡片：**
- 退会状态下，文案改为：「您已退会，原有分会绑定关系保留，仍可以非会员身份接收会议通知并参加学术会议。」

**退会后再入会完整流程**：

`
已退会状态 → 点击「重新申请入会」
    → 第 1 步：下载入会申请书模板
    → 第 2 步：上传填写好的申请书
    → 第 3 步：等待管理员审核（与首次入会一致）
    → 审核通过 → 上传会员费凭证 → 上传发票 → 点击「终审通过」→ ✓ 学会正式会员

### 需求 5：公开文件下载区

**功能说明**：新建独立的公开文件下载区，供图片大赛、短视频大赛、科普讲解大赛、党组织学习教育宣传活动等场景使用，支持四类文件格式。

#### 支持文件格式

| 分类 | 支持格式 |
|---|---|
| 文档类 | Word（.doc / .docx）、PDF（.pdf）、Excel（.xls / .xlsx）、PowerPoint（.ppt / .pptx）、压缩包（.zip / .rar） |
| 音频类 | MP3（.mp3）、WAV（.wav）、M4A（.m4a） |
| 视频类 | MP4（.mp4）、AVI（.avi）、MOV（.mov）、WMV（.wmv）、MKV（.mkv）、FLV（.flv） |
| 照片类 | JPEG（.jpg / .jpeg）、PNG（.png）、GIF（.gif）、TIFF（.tiff） |

> **说明**：原需求中「视频常用格式 MP3」为笔误，MP3 属于音频类格式，视频类不含 MP3，系统已按正确分类处理（2026-06-26 已与客户确认）。

#### 管理端（侧边栏：「内容管理 → 公开文件管理」）

管理端提供 4 个可折叠格式提示面板，每个面板包含：
- 该类型支持的所有格式扩展名
- 格式转换推荐工具及操作步骤

> 视频类 → MP4：推荐 **HandBrake（免费）** → MP4 容器 + H.264 编码  
> 文档类 → PDF：Microsoft Office / WPS → 「另存为 PDF」  
> 音频类 → MP3：**Audacity（免费）** → 「导出为 MP3」  
> 照片类 → JPEG/PNG：**IrfanView（Windows 免费）** 批量转换

| 管理操作 | 说明 |
|---|---|
| 上传文件 | 填写分类、名称、文件地址、大小、备注；弹窗内嵌格式转换建议 |
| 按分类筛选 | 切换查看文档/音频/视频/照片 |
| 逻辑删除 | 删除后前台不可见，记录保留于系统日志 |
| 下载次数 | 自动统计累计下载次数 |

#### 用户端（访问路径：/public-downloads）

- 页面顶部标注「全部文件公开下载，无需注册」
- 关键词搜索 + 四类 Tab 切换（含各类文件数量提示）
- 每个文件展示：格式标签、文件名、大小、上传日期、下载次数
- 一键下载按钮，**无需登录**

## 三、Bug 修复记录

### Bug 1：会员费发票上传后仍停留在上传界面

**现象**：在会员费缴纳步骤 4（上传电子发票），用户点击上传区选好文件并看到「电子发票上传成功！」后，再点击「提交发票，等待终审」按钮，系统却重新弹出文件选择框，且提交后不跳转。

**根因**：handleInvoiceUpload 函数调用 pickAndReadFile 再次打开文件选择框，而非直接使用已选好的文件。

**修复**：提交按钮直接使用上传区已选好的文件（memberInvoice）进行提交，不再重复弹出文件选择框。

---

### Bug 2：发票提交后会员状态未显示「发票终审中」及模拟按钮

**现象**：发票成功提交后，左侧学会会员状态仍显示「待上传发票」，看不到「⏳ 发票终审中」状态及「终审通过」模拟按钮，用户不知道还需要再操作一步。

**根因**：submitMembershipInvoice 函数调用 smartApproveInvoiceMembership，在发票提交的同时直接将状态跳转至 ctive（已激活），跳过了「发票终审中（invoice_submitted）」中间状态。

**修复**：发票提交后保留 invoice_submitted 状态，不再自动通过终审。用户可在左侧状态卡看到「⏳ 发票终审中」区域，点击「终审通过」模拟按钮后方可激活会员资格。

---

### Bug 3：新菜单未出现在侧边栏

**现象**：「新闻发布」和「公开文件管理」两个新菜单未显示在管理后台侧边栏中，无法通过导航进入。

**根因**：新增的 /admin/cms/publish 和 /admin/cms/public-files 路由未注册到 ROUTE_PERMISSIONS 权限表，也未添加到 ALL_MENU_ITEMS 菜单树，导致权限过滤后菜单项不可见。

**修复**：在 AdminContext.tsx 的权限表和菜单树中同时注册两个新路由；在 AdminSidebar.tsx 中导入 FolderUp 图标；分会管理员菜单 BRANCH_CMS_MENU_ITEMS 也同步加入「公开文件管理」入口。

---

### Bug 4：退会后「重新申请入会」按钮点击无效

**现象**：用户退会后，点击「重新申请入会」按钮，页面无任何反应，无法进入新的入会申请界面。

**根因**：hasApplied 变量包含 isWithdrawn，导致入会申请流程的显示条件 !hasApplied 始终为 alse，申请步骤界面被隐藏。

**修复**：申请流程显示条件扩展为「退会状态且用户已点击重新申请（ppFlowStep > 0）时也显示」；同时恢复按钮可点击状态，点击后直接进入申请步骤 1。

## 四、演示操作路径

### 会员费缴纳完整流程（修复后）

`
学会服务 → 会员服务 → 点击「成为正式会员」
  → 步骤 1：确认账户与金额
  → 步骤 2：银行汇款/转账
  → 步骤 3：上传凭证 → 点击「提交凭证，等待初审」→ 自动通过初审
  → 步骤 4：点击上传区选择发票文件 → 点击「提交发票，等待终审」
  → 返回主页面，左侧显示「⏳ 发票终审中」
  → 点击「终审通过」模拟按钮
  → ✓ 状态变为「✓ 学会正式会员」
`

### 退会后重新入会流程（修复后）

`
学会服务 → 会员服务 → 退会申请 → 模拟管理员通过退会
  → 左侧显示「已退会」
  → 点击「重新申请入会」按钮（蓝色，可点击）
  → 步骤 1：下载入会申请书模板
  → 步骤 2：上传填写好的申请书
  → 步骤 3：等待管理员审核（点击模拟通过）
  → 进入会员费缴纳流程（同上）
  → ✓ 重新激活会员资格
`

### 格式提示内容

**文档类**
- 支持格式：.doc / .docx（Word）、.pdf（PDF）、.xls / .xlsx（Excel）、.ppt / .pptx（PPT）、.zip / .rar（压缩包）

- 支持格式：.mp3（MP3）、.wav（WAV）、.m4a（M4A）
- 转换建议：Audacity（免费）导出 MP3；FFmpeg 提取视频音频；格式工厂批量转换

- 转换建议：HandBrake（免费）→ MP4 容器 + H.264 编码；格式工厂；ffmpeg 命令；可降低分辨率压缩体积

**照片类**
- 支持格式：.jpeg / .jpg（JPEG）、.png（PNG）、.gif（GIF）、.tiff（TIFF）
- 转换建议：Windows 画图 / Mac 预览；GIMP（免费）处理 RAW/PSD；IrfanView 批量转换

> **注**：原需求中「视频常用格式 MP3」为笔误，MP3 属于音频类格式，视频类中不含 MP3，系统已按正确分类处理。
