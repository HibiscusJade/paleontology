# 中国古生物学会网站2026年6月21日新增修改内容

> 本文档说明在 **2026年6月15日–17日** 客户需求基础上，**2026年6月21日** 对官网与管理后台高保真原型的调整内容，供客户审阅确认。  
> 对应原型：`paleontology-website-latest`（用户端）、`paleontology-admin-latest`（管理后台）。

---

## 一、官网原型：会员逻辑调整

1. **普通用户登录后选择身份（非会员 / 会员）。** 注册成功或登录后，用户需明确选择以「非会员」或「会员」身份使用网站；会员路径下还需区分学生会员、非学生会员（教师、研究人员等）。

2. **非会员与会员均可绑定学会/分会、均可报名参加学术会议。** 不再要求「必须先成为会员才能绑定、才能参会」。非会员和会员使用同一账号，均可自由绑定中国古生物学会（总学会）及任意专业分会，均可查看并报名各学会/分会发布的会议。

3. **同一会议，非会员与会员缴费金额不同。** 每场会议分别设置四类会议注册费：
   - 学生会员会议费；
   - 非学生会员会议费；
   - 学生（非会员）会议费；
   - 非学生（非会员）会议费。  
   系统根据用户当前身份自动匹配应缴金额；非会员会议费一般高于会员会议费。用户身份发生变化时，历史缴费记录仍按「当时身份」归档，便于财务对账。

4. **会员仍须走入会审核后再缴会员费。** 选择「会员」身份的用户，需下载并上传电子版入会申请书，经学会管理员审核通过后，方可缴纳会员费、成为正式会员。非会员无需缴纳会员费，可直接绑定学会/分会并报名会议。

5. **退会机制保持不变。** 有效会员申请退会须提交退会申请书并审核；退会后该用户仍以非会员身份绑定学会/分会、按非会员标准缴费参会。

## 二、官网原型：会员服务页面调整

1. **在「会员服务」中增加「中国古生物学会（总学会）」模块，置顶展示。** 与 6月15日–17日 需求一致：总学会作为独立模块放在最前面，其下依次为 11 个专业分会。总学会只发布中国古生物学术年会、重要论坛或活动，不负责发布各分会的学术年会会议；各分会仍各自发布本分会的会议。

2. **「专业分会绑定管理」更名为「会议绑定管理」。** 名称更贴近实际用途：用户绑定的是希望接收会议通知、学术资讯的学会/分会，而非单独的「分会会员身份」。

3. **会议绑定管理列表顺序。** 第一项为「中国古生物学会（总学会）」，以下依次为 11 个专业分会。用户可勾选绑定或解绑；绑定后自动接收对应学会/分会的会议通知与相关资讯；未绑定的学会/分会内容不向该用户推送。

4. **总学会会议对所有注册用户可见。** 即使用户尚未绑定任何分会，也可查看并报名总学会发布的学术年会、重要论坛等活动；绑定关系主要影响各分会会议与通知的推送范围。

## 三、管理后台原型：页面设计补充

在 6月15日 管理后台样例基础上，补充并完善业务模块与内容管理；**按角色区分可见菜单**，避免分会管理员误操作总学会内容。

### 3.1 三种管理员角色与侧栏菜单

| 角色 | 侧栏菜单（一级） | 说明 |
|------|------------------|------|
| **学会总管理员** | 首页仪表盘、审核工作台、用户管理、会议管理、数据统计、财务记录、分会管理、内容管理 | 可管理全站 12 个学会单元 |
| **分会管理员** | 首页仪表盘、会议管理、数据统计、内容管理 | **仅本分会**；不可见审核、用户、财务、分会管理等总学会功能 |
| **财务审核员** | 首页仪表盘、审核工作台、财务记录 | 专注凭证/发票及入会退会审核 |

### 3.2 学会总管理员 — 内容管理（与官网顶栏对应）

「内容管理」栏目顺序与官网顶栏一致，便于维护总学会及全站展示内容：

| 序号 | 官网顶栏 | 后台可维护内容 |
|------|----------|----------------|
| 1 | 首页 | 轮播图、新闻动态 |
| 2 | 学会简介 | 页面内容、获奖成果 |
| 3 | 组织机构 | 人员信息（理事会、常务理事会等） |
| 4 | 学会服务 | 科学传播、科技奖励 |
| 5 | 党建文化 | 党建各子栏目（12 项） |
| 6 | 学会沿革 | 历史时间线 |
| 7 | 历史相册 | 历史照片 |
| 8 | 会员公告 | 学会级公告 |
| 9 | 国际交流 | 交流动态与合作机构 |
| 10 | 资料下载 | 资料文件 |
| 11 | 规章条例 | 规章正文与附件 |

另含：**媒体库**、**站点配置**（版权、联系方式等）。

### 3.3 分会管理员 — 内容管理（仅本分站）

分会管理员 **不管理总学会首页、学会简介、组织机构人员、党建文化、学会沿革、国际交流、规章条例、站点配置** 等全站栏目，仅维护 **本分站子站点** 内容，菜单如下：

| 后台菜单 | 对应分会子站栏目 |
|----------|------------------|
| 分会栏目 | 分会概况、工作动态等页面 |
| 通知公告 | 本分站通知（附件可按会员权限控制） |
| 历史相册 | 本分站历史照片 |
| 科学传播 | 科普文章、视频等 |
| 获奖成果 | 本分站获奖信息 |
| 资料下载 | 会议简讯、摘要集、申请表等 |
| 媒体库 | 本分站图片与附件 |

所有编辑内容 **自动限定在本分会范围内**，不可查看或修改其他分会数据。

### 3.4 业务管理模块（总管理员为主）

1. **首页仪表盘** — 总注册人数、会员/非会员人数、会员费与会议费累计概况、近期缴费趋势。（分会管理员仅见本分会概况。）

2. **审核工作台** — 入会/退会申请书审核，会员费/会议费凭证与发票审核；审核结果同步至用户端。（仅总管理员、财务审核员。）

3. **用户管理** — 非会员与会员用户查询、状态管理。（仅总管理员。）

4. **会议管理** — 发布会议、配置四类会议费、上传会议通知与摘要模板、设置摘要/住宿/野外截止时间；查看单场会议参会与统计。（总管理员管全部；分会管理员仅本分会会议。）

5. **数据统计** — 全局 / 单学会 / 单会议三层统计；支持 ZIP 分类导出。（总管理员可看全部；分会管理员仅本分会。）

6. **财务记录** — 按学会查看凭证、发票并导出。（总管理员、财务审核员。）

7. **分会管理** — 维护 11 个分会基础信息。（仅总管理员。）

## 简单总结

**一、官网**  
- 登录后选择非会员/会员；两种身份均可绑定学会/分会、均可报名会议，按身份缴纳不同会议费。  
- 会员服务页顶置「中国古生物学会（总学会）」；「专业分会绑定」改为「会议绑定管理」，总学会置顶。

**二、管理后台**  
- 总管理员：全功能 + 与官网 11 顶栏对应的内容管理。  
- 分会管理员：仅「仪表盘 / 会议 / 统计 / 本分站内容管理」，**不能**编辑总学会首页、简介、党建等全站栏目。  
- 财务审核员：审核工作台 + 财务记录。

**三、与 6月15日–17日 需求的关系**  
- 6月21日修改主要为 **原型交互、角色权限与页面结构** 的调整与补充。  
- 6月15日–17日 文档中的四类会议费、凭证发票导出、入会退会审核、摘要/住宿/野外报名等规则 **继续有效**，详见《2026-06-17-客户新增修改内容-MRD》。
