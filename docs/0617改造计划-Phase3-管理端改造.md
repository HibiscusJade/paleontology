## 6. 三级统计字段清单（必须与 MRD 一致）

### 6.1 总管理员全局统计

- 总注册/会员/非会员人数（含学生/非学生分层）
- 会员费累计 + 学生/非学生笔数金额
- 全部会议费累计 + 12 学会分项

### 6.2 单个学会/分会累计

- 累计参会总人数
- 四类人群人数
- 四类会议费笔数/金额（48 项中的学会维度）

### 6.3 单次会议

- 四类费用笔数/金额、参会总人数
- 口头/展板报告数
- 住宿：总房间数、男单/男双/女单/女双
- 野外：会前/会中/会后 × 总/男/女

## 7. ZIP 导出目录规范（MRD 确认问题）

```
export_{scope}_{id}_{date}/
├── 学生会员/
│   ├── 缴费凭证/
│   └── 电子发票/
├── 非学生会员/
│   ├── 缴费凭证/
│   └── 电子发票/
├── 学生（非会员）/
│   └── ...
├── 非学生（非会员）/
│   └── ...
└── 汇总台账.csv
```

全局导出：`export_global_all_{date}/{学会名}/...`

## 9. 校验要点

### 权限

- [ ] 分会管理员登录：仅见本分会会议/统计/导出
- [ ] 总管理员：可见全部 12 学会
- [ ] 财务审核员：可访问审核/财务，不可改分会

### 统计

- [ ] 全局/学会/会议三层数字来自实收，非 registrations 估算
- [ ] 单次会议四类笔数之和 = 参会总人数（confirmed）

### 导出

- [ ] 全局/学会/会议三层均可下载 ZIP
- [ ] ZIP 内目录符合四类人群 × 凭证/发票结构
- [ ] 汇总台账.csv 字段完整

### 审核

- [ ] 入会审核通过 → 用户端可缴会员费
- [ ] 退会审核通过 → 用户端会员状态解除

### 技术

- [ ] `pnpm check` / `pnpm build` 通过
