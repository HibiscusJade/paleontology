## 7. 业务规则速查（实现时必须遵守）

| 规则 | 说明 |
|------|------|
| 总学会虚拟绑定 | `isSocietyAccessible(boundBranches, "zgswxh")` 恒 true |
| 费用锁定 | 报名时写入 `feeType`+`lockedAmount`，后续身份变更不影响 |
| 下载门禁 | 未缴费→仅公开通知；`confirmed` 后→盖章 PDF + 摘要模板 |
| 摘要 | 截止前可改；只保留最后一次 Word |
| 住宿/野外 | 默认开会前 7 天截止；截止后表单 disabled |
| 入会 | 无申请书不可提交；审核通过后才能缴会员费 |
| 退会 | 有效会员须申请书；审核通过当日终止资格 |

## 8. 校验要点（阶段完成门禁）

### 功能验收

- [ ] 总学会会议：未绑定分会也可见、可报名
- [ ] 四类费用：学生会员/非学生非会员等显示与扣费一致
- [ ] 未缴费：仅未盖章通知；缴费确认后两下载入口可用
- [ ] 摘要：截止前可替换；截止后不可改
- [ ] 住宿：男单/男双/女单/女双/自主安排 五选项可选
- [ ] 野外：会前/会中/会后多选；性别随用户 profile
- [ ] 入会：模板下载→上传→待审核→（admin 通过后）可缴会员费
- [ ] 退会：PersonalCenter 提交→待审核→（admin 通过后）会员失效

### 技术验收

- [ ] `pnpm check` 通过
- [ ] `pnpm build` 通过
- [ ] 无 deprecated `getConferenceFee(confId, userType)` 用于实际扣费
