# 中国古生物学会网站 — 快速启动指南

> **更新日期：** 2026-06-14

---

## 项目结构

```
D:\Paleontology\
├── paleontology-website-latest/   ← 前台主站 (学会门户 + 党建文化)
└── paleontology-admin-latest/     ← 后台管理系统
```

两个项目**独立构建部署**，共享 `shared/constants.ts` 中的业务常量（各项目独立拷贝）。

## 一、前台主站（学会门户）

### 1. 安装依赖

```bash
cd D:\Paleontology\paleontology-website-latest
pnpm install
```

### 2. 启动开发服务器

```bash
pnpm dev
```

访问地址：**http://localhost:3000**

### 3. 前台页面一览

| 路由 | 页面 |
|------|------|
| `/` | 首页 |
| `/intro` | 学会简介 |

| `/services` | 学会服务（会员/会议/科普/科技奖励） |
| `/personal-center` | 个人中心 |
| `/branches` | 专业分会 |
| `/party` | 党建文化（含12个子页面） |

| `/society-announcements` | 会员公告 |
| `/international` | 国际交流 |
| `/downloads-center` | 资料下载 |

## 三、联调流程

前台与后台通过共享的 `localStorage` key 联动，典型流程：

```
1. 前台注册/登录用户
2. 用户首次登录选择参与方式（非会员 / 正式会员）
3. 正式会员上传缴费凭证 → 进入初审队列
4. 后台财务审核员登录 → 审核工作台 → 初审通过
5. 用户上传发票 → 进入终审队列
6. 后台财务审核员 → 终审通过 → 会员资格生效
7. 后台总管理员 → 用户管理 → 查看/管理全部用户
```

### 共享的 localStorage Key

| Key | 用途 |
|-----|------|
| `paleo_all_users` | 全部注册用户列表 |
| `paleo_user_db` | 用户登录凭证 |
| `paleo_society_membership_{email}` | 用户会员状态与缴费记录 |
| `paleo_bound_branches_{email}` | 用户绑定的分会 |
| `paleo_confs_{email}` | 用户会议报名状态 |
| `paleo_conferences_db` | 会议配置数据 |
| `paleo_audit_log` | 审核操作日志 |
